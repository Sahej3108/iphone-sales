iPhone Sales Prediction (Flipkart Dataset)

Description:
A regression-based data science project that predicts iPhone sales based on Flipkart data such as pricing, ratings, and product features. The model helps estimate sales performance and optimize pricing strategies.

Tech Stack:

Python, Pandas, NumPy, Scikit-learn, Matplotlib

Linear Regression, Decision Tree Regressor

Key Features:

Data cleaning and preprocessing from Flipkart datasets.

Feature selection and correlation analysis for accurate prediction.

Visualization of trends and feature importance.

Results:

Achieved strong correlation between predicted and actual sales data.

Enabled better insights into pricing and customer engagement factors.

Outcome:
Delivered actionable insights into pricing impact and consumer demand forecasting.
